# Clothify Store – JavaFX POS System

Clothify Store is a standalone **Point of Sale (POS)** and **Inventory Management System** designed for clothing shops.  
This application is built using **JavaFX** for the user interface and **MySQL** for data storage.  
It focuses on real-world store operations such as product handling, stock updates, supplier records, order processing, and reporting.

---

## 🚀 Features

### 🔐 User Management
- Admin & Staff login authentication  
- Role-based access control  

### 👕 Product & Category Management
- Add, update, delete products  
- Manage categories  
- Real-time quantity updates  
- Search and filtering for fast access  

### 📦 Inventory Tracking
- Automatic stock reduction after sales  
- Low-stock alerts  
- Inventory summary view  

### 🧾 Order Processing
- Add items to cart  
- Apply quantities & discounts  
- Place orders  
- Generate invoices  
- Store complete order history  

### 🏢 Supplier & Employee Management
- Add, update, and track suppliers  
- Maintain employee details  

### 📊 Basic Reporting *(Optional Jasper Integration)*
- Daily/Monthly sales summary  
- Inventory summary report  
- Popular items  
- Transaction history
